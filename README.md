# Star Wars Example

A sample repo for deploying a microservice using k8s. This was done on macOS with docker-desktop.

## Installation

```bash
npm install
```

## Usage

```bash
npm run deploy:dev
curl http://localhost:3000
npm run undeploy:dev
```
